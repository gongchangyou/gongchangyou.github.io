1. [https://nodejs.org/zh-cn/download](https://nodejs.org/zh-cn/download) 下载最新的node
2. npm install -g openclaw
3. [https://open-claw.me/zh/guide/getting-started](https://open-claw.me/zh/guide/getting-started) 按照这个指引的第2步  ,逐步设置
	```
	openclaw onboard --install-daemon
	```

4. 直接在TUI (终端)开启对话
![]({{ site.baseurl}}/images/202602/微信图片_20260217160312_759_4.png)


5. 若两次 ctrl + C 关闭了对话框，可以通过 如下命令再次打开
```
openclaw tui
```



6. 重启openclaw
```
关闭所有：pkill -f openclaw
启动网关：openclaw gateway --token 123456
关闭网关：openclaw gateway stop
启动客户端：openclaw tui --token 123456
```
